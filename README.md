## Usage

1. create a `.env` file and add a line `DISCORD_BOT_TOKEN=<insert discord bot token here>`
2. Run `pip3 install -r requirements.txt` in the shell (navigate to the same directory as `requirements.txt` first)
3. run `main.py`
